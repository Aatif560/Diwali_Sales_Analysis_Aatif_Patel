Diwali Sales Analysis by Aatif Patel
Overview:
The Diwali Sales Analysis project involves the examination of sales
data during the Diwali festive period. This analysis aims to uncover insights 
about customer behavior, product performance, and overall sales trends. The project 
uses Python, leveraging libraries like Pandas for data manipulation, Matplotlib for basic plotting, 
and Seaborn for enhanced data visualization. The key steps include data cleaning, exploratory data analysis (EDA), 
and generating visualizations to present the findings.
